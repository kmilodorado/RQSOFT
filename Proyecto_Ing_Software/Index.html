¿Qué es RQSoft?

Es una herramienta CARE que apoya a los proyectos de desarrolo de software en la etapa de elicitación de requisitos.

Características de RQSoft

Diseño creativo

RQSoft se encuentra estructurada y diseñada bajo HTML5, CSS3, JavaScript y algunas librerias de Bootstrap, la cual permite una mayor flexibilidad, usabilidad y adaptabilidad en cualquier dispositivo.

Desarrollo web

RQSoft es una aplicación desarrollada en ambiente web bajo el entorno .NET con apoyo de librerías de Ext.NET.

Trabajo colaborativo

RQSoft ofrece a los usuarios un ambiente de trabajo colaborativo, con el que pueden estar vinculados en varios proyectos y trabajar en ellos desde cualquier lugar con acceso a internet.

El objetivo de RQSoftes facilitar la documentación de los requerimientos

  • Crear proyectos

    Con RQSoft puedes crear proyectos.

    Crear plantillas

    Con RQSoft puedes crear plantillas.

    Crear modelos

    Con RQSoft puedes crear modelos de clases, casos de uso y secuencia.
